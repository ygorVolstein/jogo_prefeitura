# **🏰 A História de Brusque 🥨**

Bem-vindo ao **A História de Brusque**\! Este é um projeto lúdico desenvolvido durante a semana da curricularização da Unifebe em parceria com a prefeitura de Brusque para celebrar e ensinar sobre a rica história da "Cidade dos Tecidos". Se você é um explorador curioso ou um desenvolvedor iniciante, este guia foi feito para você.

## **🏆 Requisitos da Missão (Entregáveis)**

Para que nossa missão seja cumprida com sucesso, precisamos entregar:

* **15 Perguntas Estruturadas:** Desafios sobre a imigração, indústria têxtil e tradições de Brusque.  
* **Protótipo Validado:** Um desenho prévio aprovado por todos.  
* **Repositório Organizado:** Pastas limpas e arquivos fáceis de encontrar.  
* **Primeira Versão Publicada:** O jogo rodando no link do GitHub Pages.  
* **Interface Simples e Intuitiva:** Qualquer pessoa deve conseguir jogar sem ajuda.  
* **Feedback Lúdico e Respeitoso:** Mensagens divertidas ao acertar ou errar as perguntas.

## **🎮 Como Usar e Modificar (Guia para Iniciantes)**

Se você quer baixar este projeto no seu computador ou fazer alterações, siga estes passos simples:

### **1\. Como baixar o projeto**

1. No GitHub, clique no botão verde **"Code"**.  
2. Escolha a opção **"Download ZIP"**.  
3. Extraia a pasta e abra o arquivo index.html no seu navegador.

### **2\. Como modificar o jogo**

* **As perguntas:** Estão no arquivo script.js. Procure pelo texto entre aspas e edite.  
* **As cores e estilo:** Estão no arquivo style.css.
